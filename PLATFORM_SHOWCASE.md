## 🎨 VISUAL DESIGN SYSTEM

### Color Palette
```
Primary Accent:    #FF7F40 (Orange)      ← Buttons, highlights, interactive
Background:        #F8F8F8 (Off-white)   ← Main background
Foreground:        #1F1F1F (Near-black)  ← Text content
Card:              #FFFFFF (White)       ← Card backgrounds
Border:            #E6E6E6 (Light gray)  ← Subtle dividers
```

### Typography
```
Display Font:  Syne (Bold, Modern)              ← Headings, buttons
Body Font:     Merriweather (Elegant, Serif)   ← Content, readable
```

### Sizing Hierarchy
```
Hero Text:        5xl-7xl        ← Large page titles
Section Heading:  4xl-5xl        ← Section headers
Subheading:       2xl-3xl        ← Component titles
Body:             base-lg        ← Regular content
Small:            sm-xs          ← Meta information
```
